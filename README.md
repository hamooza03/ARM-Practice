## ARMv7 Assembly Practice

ECSE 324 Lab - Practice converting C code to ARM Assembly.

The following files are:
- Sorting algorithms
  - Bubble sort
  - Selection sort
- Recursive code
  - Pell numbers
  - Padovan numbers
- Iterative code
  - Eratosthenes prime
  - Long division
  - RGB to grayscale
  - Pell numbers
